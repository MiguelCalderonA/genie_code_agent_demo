# Genie Code Agent Demo

A workflow demonstration showcasing the benefits of using Genie Code — Databricks AI/BI Genie integrated with Claude Code (Claude agent workflows).

## Overview

This repo walks through how Genie Code accelerates data workflows by combining:
- **Databricks AI/BI Genie** — conversational analytics over your lakehouse
- **Claude Code agents** — automated, multi-step task execution

## Structure

```
genie_code_agent_demo/
├── data/           # Sample datasets and schemas
├── notebooks/      # Databricks notebooks
├── agents/         # Claude agent workflow definitions
├── scripts/        # Helper scripts and utilities
└── docs/           # Additional documentation and diagrams
```

## Prerequisites

- Databricks workspace with Genie enabled
- Python 3.10+
- Databricks CLI configured
- Claude Code CLI
